# 🛡️ Network Penetration Test Report – Metasploitable 2

## 📌 Project Overview


Performed a comprehensive network security assessment on the Metasploitable 2 virtual machine using Nmap and manual verification techniques. The project focused on identifying open ports, vulnerable services, insecure configurations, and potential attack vectors within a controlled lab environment.


## 🎯 Objective
- 🔍 Discover open ports and running services.
- 🛡️ Identify security vulnerabilities and misconfigurations.
- ⚠️ Assess the risk of exposed services.
- 📑 Document findings with remediation recommendations.
- 💻 Gain hands-on experience in penetration testing methodology.
  
## 🧠 Skills Learned
- 🌐 Network Reconnaissance
- 🔍 Service Enumeration
- 🚪 Port Scanning
- 🛡️ Vulnerability Assessment
- 📊 Risk Analysis
- 📝 Security Reporting
- 🔐 Security Best Practices
- 💻 Linux Command Line
- 🎯 Ethical Hacking Methodology

## 🚨 Key Findings
- 🔓 Anonymous FTP Login (vsftpd 2.3.4)
- 📡 Legacy R-Services Misconfiguration
- 🚪 Telnet Service Exposure
- 📂 Samba Misconfiguration
- 💀 Unauthorized Bind Shell
- 🖥️ Weak VNC Authentication
- 🗄️ PostgreSQL Default Credentials

## 📈 Project Outcome
- ✅ Identified multiple critical vulnerabilities.
- ✅ Evaluated security risks.
- ✅ Recommended remediation measures.
- ✅ Produced a professional penetration testing report.
